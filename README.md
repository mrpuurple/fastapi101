# FastAPI 101

## Run

```sh
# run in dev mode
uvicorn --reload storeapi.main:app
# run
uvicorn storeapi.main:app
```

## Setup environment

```sh
# Create a .python-version file
pyenv local 3.11.6
# Create a virtual environment
pyenv exec python -m venv .venv
```
